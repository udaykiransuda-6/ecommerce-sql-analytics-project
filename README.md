# 🛒 E-Commerce SQL Analytics Project

## 📌 Project Overview

This project focuses on analyzing a real-world E-Commerce dataset using **SQL (MySQL)** to solve practical business problems related to:

* Revenue Trends
* Product Category Performance
* Seller Performance
* Delivery Trust Analysis
* Customer Behavior

The goal of this project is to move beyond SQL syntax and use data to solve **real business problems through analytics**.

---

## 📊 Dataset Used

Real-world Brazilian E-Commerce Dataset (Olist Dataset)

Dataset contains:

* 99,441+ Orders
* Customers
* Sellers
* Products
* Payments
* Reviews
* Delivery Information

* Monthly Revenue Trends

* Top Categories
